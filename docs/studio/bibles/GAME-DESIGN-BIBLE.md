# Game Design Bible v0.1

**Owner:** Gameplay Director  
**Status:** Draft — seeded from POS v3.0 Part I  
**Pillars served:** P1–P5 (all)

---

## 1. Vision

A 3D platformer where **movement is the primary verb** — expressive, readable, fair, and joyful. Every system exists to make jumping, exploring, and discovering feel excellent.

## 2. Principles

1. **Movement before meta** — No progression system ships until core loop Fun ≥60.
2. **Death teaches** — Every death produces data and a fast, fair retry.
3. **Discovery rewards curiosity** — Secrets are optional delight, not mandatory chores.
4. **Challenge is legible** — Players understand why they failed before they retry.
5. **Rewards are timely** — Micro-feedback within 100ms; macro-payoff every 3–5 minutes.
6. **Meta amplifies mastery** — Unlocks change how players express skill, not substitute for it.
7. **Assist without shame** — Accessibility options are first-class, not hidden.
8. **Measure before ship** — No feel change without metric or playtest evidence.

## 3. Standards

| Standard | Target |
|----------|--------|
| Core loop clarity | Player describes loop in 1 sentence (playtest) |
| Respawn time | ≤2s (≤1s post-M3) |
| Reward density | 8–15 micro-rewards/min |
| Secret taxonomy | 70% obvious / 25% attentive / 5% expert per level |
| Secondary loop gate | Disabled until M3 G1 pass |
| Fun score (internal) | ≥60 slice, ≥75 ship |

## 4. Patterns

- **Teach-before-test** — Grammar TEACH node before EXAM
- **Fail-forward** — Checkpoints, soft failure on falls
- **Express routes** — Optional mastery paths parallel to main path
- **Recover beats** — RECOVER grammar node after EXAM/boss

## 5. Anti-patterns

- Floaty, uniform movement with no state expression
- Deaths without telegraph or visible hazard
- Mandatory grind gates on star collection
- Meta progression masking weak moment-to-moment fun
- Tuning by gut without telemetry

## 6. References

- Super Mario Odyssey (expressive movement, discovery)
- Astro Bot (readability, camera, joy density)
- Celeste (fair failure, assist mode)
