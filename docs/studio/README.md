# Studio Operating Layer (SOL)

The **Studio Operating Layer** is subordinate to the [Game Design Intelligence Layer](../gdil/GAME-DESIGN-INTELLIGENCE-LAYER.md) and governs the [Production Operating System](../phases/PRODUCTION-OPERATING-SYSTEM.md). GDIL ensures design quality; SOL ensures studio operations; POS defines engineering work.
